## Task Management Dashboard
##### Features :
1. Task Functionality - 
- Add a task 
- Edit a task 
- Delete a task
- Mark task as completed 

2. Filters - 
- All tasks
- Completed tasks 
- Pending tasks
- Overdue tasks

3. Search functionality - 
- Search for a task using title 


 Technology Used - 
- Redux toolkit
- Reactjs 
- TailwindCss

  
Deployed Link : https://task-manager-dash-g4y4.vercel.app/

Run locally using Vite :
- npm install
- cd frontend 
- npm run dev
